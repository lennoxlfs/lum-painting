## Monthly SEO evolution add-on
If a client is on the monthly SEO evolution package, invoke the `monthly-seo-evolution` skill every month or whenever monthly analytics/reporting work is requested.

For monthly SEO evolution clients, treat analytics and search data as the source of truth for optimization priorities.
Always produce:
1. a client-facing monthly SEO evolution report
2. an internal implementation report
3. a ranked list of approved website changes to implement next

Do not treat monthly SEO evolution as passive reporting.
It must produce action items, implementation guidance, and measurable next-month priorities.
