# MONTHLY ANALYTICS STACK FOR LFS SEO EVOLUTION

## Install / configure on every Vercel site
1. Vercel Web Analytics
2. Vercel Speed Insights
3. GA4
4. Google Search Console Domain property
5. Bing Webmaster Tools
6. Google Business Profile access for local businesses
7. Rich Results Test in QA workflow
8. Lighthouse / PageSpeed on priority pages each month

## GA4 recommended events
- page_view
- generate_lead
- contact_form_submit
- phone_click
- email_click
- book_now_click
- appointment_request
- quote_request
- map_directions_click
- scheduler_complete
- chat_start

## Monthly pull list
### Google Search Console
- Search results performance export
- top queries
- top pages
- query/page CTR opportunities
- indexing / coverage notes if relevant
- Search Console Insights notes if used

### GA4
- landing page report
- traffic acquisition report
- key events / conversion report
- engagement by landing page
- referral traffic notes

### Vercel
- Web Analytics overview
- Speed Insights overview
- page-level performance notes

### Local businesses
- GBP performance notes
- reviews / photo freshness
- hours / service info accuracy

### Search / crawl / QA
- Lighthouse mobile + desktop on homepage + top service page(s)
- PageSpeed Insights homepage + top service page(s)
- Rich Results Test on homepage + key service/location pages
- robots.txt / sitemap.xml check

- Lighthouse mobile homepage

- Lighthouse desktop homepage

- PageSpeed mobile homepage
