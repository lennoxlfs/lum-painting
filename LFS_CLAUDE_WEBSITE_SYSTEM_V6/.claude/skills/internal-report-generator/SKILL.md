# Internal LFS Report Generator

Use this skill when preparing the internal-only strategy and sales notes for the meeting.

## Inputs
- current website URL
- sample site URL or summary
- Research Package
- Brand Package if relevant
- user notes
- likely package options

## Required sections
1. Prospect Overview
2. Real Findings
3. What the Sample Fixed
4. Talking Points
5. Close Strategy

## Talking points should include
- top visual wins
- top SEO wins
- top conversion wins
- low-risk trust points
- likely objections
- best responses

## Close strategy should include
- default package to pitch
- whether to mention care now
- whether to mention growth now
- whether the 90-day test should stay off the table or be used as a rescue close

## Tone
- direct
- strategic
- concise
- more technical than the client report
