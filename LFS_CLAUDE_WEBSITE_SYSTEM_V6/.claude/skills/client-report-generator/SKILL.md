# Client Report Generator

Use this skill when generating the report that will be handed to the business owner.

## Inputs
- current website URL
- sample site URL or sample summary
- Research Package
- key visible wins from the sample

## Goal
Sell the value clearly without giving away the proprietary implementation playbook.

## Required sections
1. Business Snapshot
2. Current Site Issues
3. Issue-by-Issue Comparison
4. Why the New Version Performs Better
5. Recommended Next Step

## Comparison style
For each issue, use a simple pattern:
- Issue
- Current Site
- Our Version

## Tone
- simple
- professional
- confident
- easy to scan
- business-owner friendly

## Do not include
- full technical audit detail
- exact SEO framework
- exact implementation map
- enough detail for another provider to copy the strategy for free

## Preferred behavior
Explain improvements in outcome language:
- clearer structure
- easier navigation
- stronger local visibility foundation
- stronger trust
- cleaner customer action flow
- stronger overall business presentation
