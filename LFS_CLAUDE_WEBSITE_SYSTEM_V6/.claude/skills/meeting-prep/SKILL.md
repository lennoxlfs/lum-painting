# Meeting Prep

Use this skill when the user wants to prepare for Step 5.

## Inputs
- client-facing report
- internal LFS report
- sample site
- package pricing
- meeting context if any

## Output structure
1. Meeting Flow
2. Opening Framing
3. Key Visual Walkthrough Points
4. Key Business Outcome Points
5. Objections to Watch For
6. Close Sequence
7. Follow-Up Plan if Not Closed

## Rules
- keep the flow simple
- lead with visible wins
- connect those wins to business outcomes
- protect the proprietary method
- keep the 90-day test as a conditional closing tool, not the default offer
