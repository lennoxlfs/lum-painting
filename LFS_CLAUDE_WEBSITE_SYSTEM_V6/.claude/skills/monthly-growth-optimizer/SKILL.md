# Monthly Growth Optimizer

Use this skill after launch when analytics or search data are available.

## Typical inputs
- Search Console Performance summary or export
- Search Console Insights summary
- page indexing / URL inspection notes if relevant
- GA4 Landing Page report
- GA4 engagement metrics
- key events / conversion summaries
- GBP Performance summary
- Lighthouse or PageSpeed reruns if relevant
- current site URL
- priority business goals

## Objectives
- identify weak pages
- identify visibility opportunities
- identify CTR opportunities
- identify engagement leaks
- identify conversion leaks
- identify local-profile opportunities
- recommend or implement focused changes
- keep the site improving over time

## Use this workflow
1. What the data is saying
2. Priority Issues
3. Exact Changes to Make
4. Expected Impact
5. Next-Month Focus

## Rules
- separate measured conclusions from reasonable inference
- prioritize issues with clear business value
- do not confuse maintenance with growth work
- use `reference/monthly-growth-loop.md`
- prefer small, high-impact, evidence-based changes over random rewrites
- tie each change to a metric or business goal
