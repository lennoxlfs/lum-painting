# Brand Package Builder

Use this skill when the user wants the Step 2B Brand Package.

## Primary input
- current website URL

## Goal
Extract enough brand identity from the current site to preserve or elevate recognizable brand DNA in the redesign.

## Always attempt to infer
- business name
- logo traits
- visible logo/image references
- photo style
- brand colors
- likely font direction
- button style
- spacing feel
- layout style
- visual tone
- copy/tone direction
- what should be preserved vs modernized

## Output structure
1. Brand Snapshot
2. Asset Inventory
3. Color Palette
4. Font / Style Direction
5. Layout & UI Notes
6. Tone of Voice
7. Brand DNA Summary
8. Redesign Guardrails

## Rules
- default to preserving recognizable brand DNA unless directed otherwise
- describe fonts by exact name only if clearly known; otherwise describe style direction honestly
- do not force your own visual style onto the brand
- focus on what will help the sample site feel familiar but upgraded
