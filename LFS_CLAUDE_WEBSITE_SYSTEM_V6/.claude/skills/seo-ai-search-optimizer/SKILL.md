# SEO + AI Search Optimizer

Use this skill when the user wants a page plan, content plan, site build, or audit aligned with current Google guidance.

## Inputs
- current site or draft site
- target services
- target geography
- Research Package if available
- Brand Package if available

## Main objectives
- satisfy Google Search technical basics
- strengthen helpful, people-first page quality
- strengthen local entity and business details
- support Google AI features through foundational SEO, strong textual clarity, unique value, and useful media
- support Bing and OpenAI search discoverability without spammy tactics
- avoid thin, repetitive, or commodity pages

## Use these references
- `reference/google-search-playbook-2026.md`
- `reference/google-ai-search-playbook-2026.md`
- `reference/local-seo-and-entity-playbook.md`
- `reference/ai-search-and-openai-discoverability-playbook-2026.md`

## Output structure
1. Core SEO risks
2. Core AI-search risks
3. Priority fixes
4. Content / structure recommendations
5. Local-business recommendations
6. What to measure after launch


## OpenAI search rule
Where client goals allow it, do not accidentally block OAI-SearchBot in robots.txt.
Support discoverability with strong crawlable HTML, clear answers, internal links, and measurable referral tracking.
