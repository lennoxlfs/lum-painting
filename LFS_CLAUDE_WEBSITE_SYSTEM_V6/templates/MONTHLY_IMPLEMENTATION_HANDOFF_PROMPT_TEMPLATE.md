# Monthly Implementation Handoff Prompt Template

Read `monthly-internal-implementation-report.md` before making changes.
Use it as the direct implementation brief for this month’s approved updates.

Requirements:
- implement only the approved changes
- prioritize SEO, internal linking, content clarity, schema, performance, accessibility, and conversion improvements as outlined
- preserve the site’s working design unless a visual change is explicitly approved
- update `implementation-log.md`
- prepare the project for QA, GitHub push, and Vercel deployment
